### 🛠️ Stack Tecnológico
* **Data & BI:** Power BI, Looker Studio, SQL, Google Cloud.
* **Programación:** Python (Automatización y análisis).
* **Web:** HTML5, CSS3 (Diseño responsivo y animaciones integradas).
* **Sistemas de Gestión:** ISO 45001:2018, SGMM.
